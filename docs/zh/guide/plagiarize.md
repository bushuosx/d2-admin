# 借鉴者和抄袭者

以下开源项目中存在 D2Admin 代码

* 借鉴：少量复制代码，并明确放置 D2Admin 项目链接

* 抄袭：大量复制

**D2Admin 是作者从2018年1月就开始开发的心血，抄袭是很不道德的行为，如果你也发现了其它框架采用了本项目的代码和设计，欢迎联系我**
